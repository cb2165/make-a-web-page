HTML-Hypertext Markup Lanuage

Is the basics of all web pages and will include the following:

  • Text content
  • Markup (what it looks like)
  • Images and videos
  • Other documents
  • Links to other pages


What is an element?

An elment consists of a Name (opening tag) then the contents followed by a closing tag.

Basic Tags

Inline

  • b=Bold
  • em=Italics
  • a=Anchor (add hyperlinks, needs attribute like href)
  • img=Image(no closing tag, needs source and alt txt)
  • br=Line break (no closing tag needed)
  • q=Shourt quotation
  • strong=Defines important txt
Block
  • P=paragraph
  • body=body

Note: above is just a sample of Tags to use. Can use HTML site: href="http://www.web-source.net/html_codes_chart.htm."

What is CSS?

Cascading Style Sheets-used for describing the look and formatting of a document written in a markup language.

What is DOM?

Document Object Model-a cross-platform and language-independent convention for representing and interacting with objects in HTML (and other markup languages). The node of every document are organized in a tree structure, called the DOM tree
